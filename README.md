# ACDSFM - Rádio e TV ao vivo

Projeto em HTML, Node.js, Socket.IO e WebRTC.

## Recursos

- Página principal com nome ACDSFM
- Indicador AO VIVO no topo
- TV ao vivo centralizada
- Chat ao vivo ao lado direito
- Painel admin protegido por senha
- Transmissão por webcam e microfone

## Rodar localmente

```bash
npm install
npm start
```

Acesse:

- Site: http://localhost:3000
- Admin: http://localhost:3000/admin

Senha padrão:

```txt
123456
```

No Render você pode trocar a senha criando a variável de ambiente `ADMIN_PASSWORD`.
